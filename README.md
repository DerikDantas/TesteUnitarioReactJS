## 💻 Projeto

## 🔧 Instalação
Para executar o projeto em sua máquina/dispostivo, serão necessários os seguintes recursos instalados:

- Node.js
- Yarn (Opcional)

Clonar o projeto
```console
$ git clone https://github.com/DerikDantas/TesteUnitarioReactJS.git
```
Instalar npm
```console
$ npm install
```
Rodar projeto
```console
$ npm start
```

## Para Rodar os Testes
```console
$ npm test
```
